Unity之UGUI框架

发表于2023-05-02|更新于2023-05-12|Unity3DCsharp功能框架GUI

前言

UI是一个游戏必不可少的一部分。对于一个游戏系统而言，它的UI也是多种多样的。经常地，因为要满足不同的需求，当前场景中的UI也会频繁切换。因此，为了更方便的管理这些UI的状态和行为，开发一个基于UGUI系统的UI管理框架。

一、需求分析

  1. 进入游戏时，首先会加载主 UI ，主 UI 随着游戏的进入和退出而加载和销毁。
  2. 通过一些按钮，我们可以打开其他的 UI 界面，同时主 UI 不会消失
  3. 当前场景可以同时存在多级 UI ，只有处于顶级的 UI 才能被选中并进行操作
  4. 除了主 UI 外的其他 UI 均可以在游戏场景未关闭下，手动关闭
